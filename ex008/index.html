Principais formatações

Negrito / Destaque

Existem duas maneiras de se colocar um texto em negrito, nesse trecho estamos usando a tag B, e essa é uma forma é não semântica

Nessa forma temos um termo em destaque e essa forma é semântica

Itálico / Ênfase

De mesmo modo existem duas formas de se colocar um texto em Itálico. Uma delas é usar a tag I para destacar um determinado termo essa forma não é semântica

A outra é usar a tag EM (semântica) para criar um termo em destaque

Texto marcado

Podemos marcar um texto usando a tag MARK

Texto grande e texto pequeno

Estamos criando um texto grande e um texto pequeno

Texto excluído

Podemos marcar um texto como excluído para indicar que ele deve ser lido mas não considerado

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi inserido depois
